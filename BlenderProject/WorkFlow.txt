APUNTES ANIMACION BLENDER
- SACAR UN MODELO DE UN SKELETON -> ALT+P (keep transform)
- SIEMPRE, AL USAR EL RIG DEL DARIO, MALLA POR PARTES, NUNCA HACER JOINT QUE SE JODE

WORKFLOW PARA LOS MODELOS
- PARA SACARLOS DEL ESQUELETO -> ALT+P (keep transform) (los del proyecto ya estan todos sacados)
- ELMINAR SU MODIFICADOR DE ARMADURA (los del proyecto ya estan todos sacados)
- DUPLICALO PARA TENER LA COPIA PARA OTROS MODELOS
- OPCIONAL (AÑADIR OBJETOS EXTRA), SITUALOS DONDE QUIERAS PERO NO HAGAS JOINT, DEBEN IR SEPARADOS
- ANTES DE EMPARENTAR, DUPLICAR EL ROOT CON EL ESQUELETO PARA EL SIGUIENTE
- PARA EMPARENTAR, SELECCIONAR TODOS LOS ELEMENTOS DEL PERSONAJE Y POR ÚLTIMO EL ESQUELETO -> CTROL+P (pesos automáticos)
- REZAR
- RETOCAR LOS PESOS SI ES NECESARIO
- SI LOS ELEMENTOS NUEVOS SE CHAFAN -> SELECCIONAS EL ELEMENTO COMO OBJETO, MODO EDICIÓN (seleccionas la maya entera), 
  VAS A (Object Data Properties) El triangulo encima de los materiales
  ELIGES EL HUESO QUE QUIERES QUE MUEVA EL OBJETO, EJEMPLO, GORRO -> CABEZA, (weight = 1 y Assign) DESPUÉS EL RESTO DE HUESOS QUE HAYA ALREDEDOR QUE PUEDAN INFLUIR
  (weight = 0 y Assign)
- IMPORTANTE -> PARA CADA NUEVO MODELO, CREAR UNA COLECCION Y DUPLICAR AHÍ EL ROOT_BASE CON SU ESQUELETO_BASE
